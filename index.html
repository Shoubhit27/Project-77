Project on console

Visual Studio Code is a code editor redefined and optimized for building and debugging modern web and cloud applications.Visual Studio Code is a source-code editor that can be used with a variety of programming languages, including Java, JavaScript, Go, Node.js, Python and C++.Visual Studio Code can be extended via extensions, available through a central repository.
Setting up and running with Visual Studio Code is quick and easy, it is a small download so you can install in a matter of minutes.


Console window is very useful for interacting with and debugging your code. In the console window you can enter any javascript code and methods and test your code locally.
You can open the console window by pressing ctrl+shift+j in chrome and ctr+shift+k in firefox or left mousebutton + inspect + console
For printing on console screen use 'console.log()'
